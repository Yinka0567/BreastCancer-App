# BreastCancer-App
## A Breat Cancer Prediction App built Python and Streamlit
The app uses a Random Forest Classifier trained on real breast Cancer data to predict whether a tumor is Benign or Malignant.
### Key Steps:
- Cleaned and preprocessed the dataset
- Selected the top 10 most important features using model-based feature importance.
